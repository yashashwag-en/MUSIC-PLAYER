# 🎵 My Music Player

A simple music player built using **HTML, CSS, and JavaScript**.

This project started with a pretty simple idea—I wanted one place where I could keep my favorite songs and instantly jump into them whenever I felt like listening. Instead of opening different apps or searching for playlists, I thought, *why not build my own music player?*

So I started with the basics: plain HTML, CSS, and JavaScript. No frameworks, no libraries—just the core technologies that power the web.

## ✨ Why I Built This

I've always enjoyed listening to music while studying, coding, or just relaxing. I wanted a lightweight player that contained **my personal collection of favorite songs** and gave me complete control over the experience.

This project is also a way for me to improve my frontend development skills by building something I would actually use.

Right now, I've added a few of my favorite songs to get started, and I plan to keep adding more over time. You can think of this project as a playlist that's growing along with me.

## 🚀 Features

* ▶️ Play and Pause music
* ⏮ Previous & Next song controls
* 🎶 Displays song title and artist
* ⏱ Progress bar with seeking
* 🔊 Volume control
* 📃 Playlist support
* 🔁 Autoplay functionality
* 📱 Responsive design for different screen sizes
* 🎨 Clean and modern user interface

## 🛠️ Built With

* HTML5
* CSS3
* Vanilla JavaScript
* HTML5 Audio API

## 💡 What I Learned

Building this project helped me understand how audio is handled in the browser and gave me hands-on experience with JavaScript event handling, DOM manipulation, responsive UI design, and creating a smoother user experience through small interactions and animations.

It also reminded me that sometimes the best projects are the ones you build for yourself.
